# Mangalam HDPE Pipes – Product Landing Page

This project is a responsive product landing page built using **vanilla HTML, CSS, and JavaScript**.  
It was developed as part of a frontend assignment to demonstrate clean UI implementation, responsive design, and interactive components without using any frameworks.

## Features

- Fully responsive layout (Desktop, Tablet, Mobile)
- Sticky header that appears after scrolling past the hero section
- Image carousel with thumbnails
- Hover zoom preview for carousel thumbnails
- Mobile navigation drawer with hamburger menu
- Interactive FAQ accordion
- Application and testimonial carousels
- Clean and modern UI based on provided design

## Technologies Used

- HTML5 (Semantic structure)
- CSS3 (Flexbox, Grid, responsive design)
- Vanilla JavaScript (DOM manipulation and interactions)
